FixedPointAccelerationNext.jl is a package for solving fixed-point problems using various acceleration methods. It is a fork of [FixedPointAcceleration.jl](https://github.com/s-baumann/FixedPointAcceleration.jl) and aims to be faster and new API opionated version of the original package. The package provides implementations of several acceleration methods, including Anderson Acceleration, Broyden's Method, and Aitken's Delta-Squared Process.

The implementation is inspired and based on the work of Stuart Baumann in [FixedPointAcceleration.jl](https://github.com/s-baumann/FixedPointAcceleration.jl).
